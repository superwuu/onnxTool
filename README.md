##### 代码结构

​	结构结构

<details open>
<summary>Linux</summary>
`Linux` 系统命令如下`（cmake＋make）`

```bash
# 创建build空间
mkdir build && cd build
# cmake构建
cmake ..
# make编译
make
# 构建库文件
cd .. && bash install.bash
```

</details>

<details open>
<summary>Windows</summary>

`Windows` 系统命令如下`（cmake＋msbuild）`

```bash
# 创建build空间
mkdir build && cd build
# cmake构建
cmake ..
# make编译
make
# 构建库文件
cd .. && bash install.bash
```
</details>

执行完成后，在build目录下出现`include`和`lib`两个目录，即onnxTool工具库
